# Small Happy Ghast

A lightweight, server-side-only Fabric mod that makes the Ghastlings grow only when fed a snowball. Designed for minimal impact.

## Features

- The Ghastlings will **not** grow automatically, but only grows after being fed a snowball.
- Fully server-side only.

## Installation

1. Download the latest release of **Small Happy Ghast** (`.jar` file).
2. Place it into your Fabric server's `mods` folder.
3. Launch **Minecraft 1.21.6+** with Fabric Loader.
